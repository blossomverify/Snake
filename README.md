# Snake Game

A classic Snake Game implementation developed in Python.

## Description
This project is a recreation of the classic arcade game Snake. The objective is to navigate the snake to collect food, which incrementally increases both the player's score and the snake's length. The challenge scales as the game progresses, requiring quick reflexes to avoid colliding with the screen boundaries or the snake's own body.

## Features
- **Classic Mechanics:** Fluid snake movement and growth.
- **Score Tracking:** Real-time scoring system.
- **Collision Detection:** Accurate boundary and self-collision detection resulting in game over.
- **Responsive Controls:** Intuitive keyboard-based navigation.

## Prerequisites
- Python 3.x

## Installation & Execution

1. Clone the repository:
   ```bash
   git clone https://github.com/blossomverify/Snake.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Snake
   ```

3. Execute the main script:
   ```bash
   python main.py
   ```
